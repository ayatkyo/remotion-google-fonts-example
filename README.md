## Generating Fonts .ts

```console
pnpm generate-fonts
```

All fonts will be generated to `./src/fonts`

## Commands

**Install Dependencies**

```console
pnpm i
```

**Start Preview**

```console
pnpm start
```

**Render video**

```console
pnpm build
```

**Upgrade Remotion**

```console
pnpm run upgrade
```
